# Garantias Institucionais

## Garantias que o sistema busca representar

- autoria rastreavel
- pertencimento territorial
- elegibilidade verificavel
- deliberacao com quorum
- historico de alteracao normativa
- consolidacao auditavel

## Garantias ja presentes no alpha

- rastreabilidade de proposta para commit
- restricao territorial de voto
- restricao de cidadania para acoes materiais
- historico local de alteracoes normativas

## Garantias ainda incompletas

- autenticacao forte por assinatura
- verificabilidade protocolar do voto
- imutabilidade da consolidacao normativa
- formalizacao de autoridade institucional
- trilha de parecer como entidade propria

## Principio de evolucao

Cada migracao para on-chain deve reforcar uma garantia institucional clara, e nao apenas adicionar complexidade tecnica.
