# Alpha

## Objetivo

Validar o motor legislativo versionado em ambiente controlado.

## Caracteristicas do alpha

- frontend React funcional
- API local Express
- persistencia em JSON
- cidadania local
- propostas
- votos ponderados
- comentarios
- variacoes locais
- contratos locais deployaveis

## Debitos conhecidos

- autenticacao fraca
- sync oportunistica
- review submodelado
- divergencia entre backend e protocolo

## Criterio de sucesso

Demonstrar que a logica institucional do produto faz sentido mesmo antes da convergencia on-chain completa.
